# urlcv/email-finder

Generate search engine queries to find professional email addresses by name and company.

> **Live demo:** [urlcv.com/tools/email-finder](https://urlcv.com/tools/email-finder)
> This package powers the free Email Finder tool on **[URLCV](https://urlcv.com)** — the recruitment platform that helps agencies present candidates professionally.

---

## Features

- **Input:** Full name and company (optional: company domain)
- **Output:** Pre-built search URLs for Google, Bing, and DuckDuckGo
- Multiple query strategies: direct email search, LinkedIn, contact pages, site-specific
- Fully client-side — no server round-trip, no API keys
- Copy links or open in new tab

---

## Requirements

- PHP **8.1** or higher
- Laravel (for Blade views and tool integration)

---

## Installation

```bash
composer require urlcv/email-finder
```

---

## Use cases for recruiters

- Finding candidate contact details when you only have name and employer
- Sourcing outreach when LinkedIn InMail credits are exhausted
- Verifying email addresses before cold outreach
